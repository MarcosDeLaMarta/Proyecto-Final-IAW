# Proyecto-Final-IAW

## Funcionalidades de la aplicación:

### Listado de productos
- Cuando un usuario inicie sesión, se mostrará un listado de productos.

### Ver un producto
- Al hacer clic en un producto, se muestra una vista detallada del mismo con su precio y una pequeña descripción.

### Inicio de sesión
- En la base de datos existen dos usuarios: admin y marcos (ambos con la contraseña 1234). Al iniciar sesión con el usuario marcos, se mostrará un listado de los productos. En cambio, al iniciar con el usuario admin, se mostrará un listado de los productos con la opción de añadir un nuevo producto a la tabla.

### Dar de alta un nuevo producto
- En la vista del administrador, hay un botón para añadir un nuevo producto a la base de datos.

### Añadir producto al carro de compra y ver carro de la compra
- Cuando un usuario le de a "añadir al carrito" en un producto, este se añadirá a un carrito de compra en el cual podrá borrarlo si así lo desea.
